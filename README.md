# Zero to Kernel

A 5-month self-directed curriculum working down the systems
stack, from distributed backend engineering to OS internals, culminating in a toy OS written in Rust.

Built by someone who wanted to understand how computers
actually work, not just how to use them.

## Goal

Most engineers work at one layer of the stack their entire
career. This is an attempt to understand all of them —
from distributed services down to bare metal — and build
something real at each level.
